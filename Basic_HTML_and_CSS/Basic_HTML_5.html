Automated HTML and CSS Course Challenge

Define and learn how to use the following tags:

  1. <article>
  2. <details>
  3. <summary>
  4. <time>
Tag Definitions <article>

Specifies independent, self-contained content.

<details>

Specifies additional details that the user can open and close on demand.

<summary>

Defines a visible heading for the <details> element.

<time>

Defines a specific time, such as Tech Tech Academy oprating hours are from 0900 to 1800.
